# 🏏 Bat Ball Stump Game

A simple and interactive Bat Ball Stump game built using HTML, CSS, and JavaScript. Choose Bat, Ball, or Stump and try to beat the computer. The game randomly selects the computer's move and displays the result instantly.

## 🚀 Features

- Choose between Bat, Ball, and Stump
- Random computer move generation
- Instant result display (Win, Lose, or Tie)
- Reset button to start a new game
- Simple and responsive user interface

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🎮 How to Play

1. Click on Bat, Ball, or Stump.
2. The computer will randomly choose its move.
3. The result will be displayed instantly.
4. Press the **Reset** button to start again.

## 🎯 Purpose

This project was created to practice JavaScript concepts such as:
- Functions
- Conditional Statements
- Random Number Generation
- DOM Manipulation
- Event Handling
